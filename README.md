# Andrea Cimmino — Full-Stack Software Engineer

A modern, high-performance portfolio website showcasing full-stack development expertise with the MERN stack (MongoDB, Express, React, Node.js).

## 🎯 About

I'm a full-stack software engineer specialising in building scalable, production-ready applications. This portfolio demonstrates my work across end-to-end application development, from responsive frontends to well-architected APIs and reliable data layers.

## ✨ Features

- **Responsive Design** — Works seamlessly across desktop, tablet, and mobile devices
- **Modern Animations** — Smooth scroll-triggered animations using Framer Motion
- **Dark Theme UI** — Eye-catching gradient backgrounds with glassmorphism effects
- **Fast Performance** — Optimized images, lazy loading, and code splitting
- **Accessible** — Semantic HTML, ARIA labels, keyboard navigation
- **Security-First** — HSTS, CSP, XSS protection, and MIME sniffing prevention
- **SEO Ready** — Proper meta tags, structured markup, fast Core Web Vitals

## 🛠️ Tech Stack

**Frontend:**
- React 19.2.4 — Modern component library
- Next.js 16.2.0 — Framework with App Router
- TypeScript — Type safety and better IDE support
- Tailwind CSS 4 — Utility-first styling
- Framer Motion 12 — Smooth animations

**Development:**
- ESLint — Code quality and consistency
- PostCSS — Modern CSS transformations

## 🔒 Security

This project includes modern security best practices:

- **HSTS** — Forces HTTPS connections
- **Content Security Policy** — Prevents XSS and injection attacks
- **Permissions Policy** — Disables unnecessary browser features
- **X-Frame-Options** — Prevents clickjacking
- **No hardcoded secrets** — All sensitive data kept out of version control
